# Documentación (DECIES Platform)

Este repositorio contiene la implementación técnica de la plataforma DECIES.
La documentación incluida aquí es principalmente técnica: arquitectura, stack, calidad, sprints y registros de decisiones.

## Estructura

* `technical/`

  * `02_diseno_sistema/`
    Diseño del sistema: entidades, arquitectura lógica, flujos y casos de uso.
  * `04_stack_tecnico/`
    Stack técnico ejecutable: API, base de datos, DDL, seeds, diagramas.
  * `05_calidad/`
    Plan de testing y validación.
  * `06_sprints/`
    Plantillas y desglose de sprints.

* `07_gobierno/`
  Registro de decisiones (ADRs), supuestos y documentación de CI/DB relacionada con el gobierno técnico del proyecto.

* `product/`
  Resumen público del producto (sin documentación estratégica o de marca).

## Cómo empezar (desarrollo local)

Consulta el README principal del repositorio y los documentos en `technical/04_stack_tecnico/` para levantar el entorno con Docker y ejecutar el backend.
